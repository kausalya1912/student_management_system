# 📌 Internship Management System

A simple Python-based Internship Management System that helps manage internship records using JSON file storage.

---

## 🚀 Features

- Add new internship records  
- View all internship details  
- Update existing internship information  
- Delete internship entries  
- Data stored in `internships.json` file  
- Simple command-line interface (CLI)

---

## 🛠️ Tech Stack

- Python 🐍  
- JSON (for data storage)  
- File Handling
